# Worksheet Generator

This repo contains a React application to generate printable worksheets for kids to practice.

## Getting Started

### Install Dependencies
```bash
npm install
```

### Development
```bash
npm run dev
```

### Build for Production
```bash
npm run build
```

### Run Linter
```bash
npm run lint
```

## Features
- Interactive UI for selecting worksheet types (Addition, Subtraction, Multiplication, Division)
- Modern React setup with Vite for fast development
- Built-in ESLint configuration for code quality
